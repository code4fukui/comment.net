# comment.net

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

comment.net is an open-source anonymous comment platform that allows users to post comments and share them in real-time.

## Demo
[comment.net](http://コメント.net)

## Features
- Real-time comment display
- Anonymous commenting
- Ability to reply to comments
- Customizable room creation
- View-only mode for projection

## Usage
1. Enter a room ID and password (if required) to create or join a room.
2. Type a comment in the input field and click "Send" to post.
3. Comments will be displayed in chronological order.
4. Click the "Reply" link below a comment to reply to it.
5. Use the "Projection View" and "Copy All Comments" links to access additional features.
