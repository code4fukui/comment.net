# コメント.net

コメント.net は、会議やイベントの参加者が匿名でコメントを投稿し、リアルタイムに共有できるアプリケーションです。

## デモ
[https://コメント.net/](https://コメント.net/) にてデモを体験できます。

## 機能
- 匿名でのコメント投稿
- コメントの返信機能
- リアルタイムでのコメント表示
- 投影用ビューページ
- コメント全文のテキストコピー

## 必要環境
特に必要なものはありません。ウェブブラウザがあれば利用できます。

## 使い方
1. ブラウザで [https://コメント.net/](https://コメント.net/) にアクセスします。
2. 任意のルーム名を入力し、「はいる」ボタンをクリックします。
3. コメントを入力し、「送信」ボタンをクリックすると、コメントが投稿されます。
4. 投稿されたコメントは即座に表示されます。
