# comment.net

comment.netは、会議やプレゼンテーション、イベント中のリアルタイム共有を目的とした、オープンソースの匿名コメントプラットフォームです。

## デモ
[comment.net](http://コメント.net)

## 機能
- **リアルタイム表示**: コメントはすべての参加者に即時に表示され、最新のコメントが先頭に並びます。
- **匿名投稿**: ログインや本人確認なしにコメントを投稿できます。
- **返信機能**: 特定のコメントに返信し、スレッド形式の会話を構築できます。
- **カスタムルーム**: 一意で共有可能なコメントルームを作成できます（ルーム作成にはパスコードが必要です）。
- **プロジェクションモード**: プロジェクターや大画面での表示に最適化された、拡大フォントの表示専用モードです。
- **簡単なエクスポート**: 「すべてのコメントをコピー」機能で、会話全体をプレーンテキストとしてエクスポートできます。
- **QRコード共有**: ルームURLのQRコードを自動生成し、参加者のスムーズな参加を支援します。

## 使い方
1. **ルームの作成**: メインページで希望するルーム名と作成パスコードを入力し、「はいる」をクリックします。
2. **ルームへの参加**: ルームのURL（例: `http://コメント.net/#your-room-name`）にアクセスします。
3. **コメントの投稿**: 入力フィールドにテキストを入力し、「送信」をクリックします。
4. **ツールの利用**: コメントリスト下部のリンクから「プロジェクタービュー」や「すべてのコメントをコピー」機能にアクセスできます。

## 開発者向けツール

### コメントのCSVエクスポート
ルームからすべてのコメントを取得し、`comments.csv`として保存するDenoスクリプトが含まれています。

ターミナルから以下のコマンドを実行します:
```sh
deno run --allow-net makecommentscsv.js [room-id]
```

## 技術的詳細
- 完全にウェブブラウザ上で動作するクライアントサイドアプリケーションであり、デプロイにあたってサーバーサイドの設定は不要です。
- リアルタイムデータの永続化と取得には、外部の`commgate` API（`http://sabae.club/commgate/1/`）に依存しています。
